How to Use Our API:

  • Access the bank list endpoint with a GET parameter branch_name:
    • http://127.0.0.1:8000/banks/?branch_name=< Branch Name >
    eg : http://127.0.0.1:8000/banks/?branch_name=RTGS-HO

  • Access the branch details endpoint with the IFSC code:
    • http://127.0.0.1:8000/branches/?ifsc_code=< IFSC code >
    eg : http://127.0.0.1:8000/branches/?ifsc_code=ABHY0065002
